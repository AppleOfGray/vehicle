# 车联网，路线规划
+ vehicle-admin: 运营后台
+ vehicle-center: 核心业务
+ vehicle-domain: 领域模型(表结构等)
+ vehicle-web: C端web层
+ nacos: nacos本地后台服务
+ other: 其他信息(mysql数据库表结构)

# 涉及技术
 + SpringBoot 底层平台
 + SpringCloud(alibaba) 微服务
 + openFeign 微服务调用技术
 + Nacos 服务发现配置管理
 + Mysql 数据库
 + MybatisPlus 半自动ORM
 + Swagger 后台接口自动生成
 + sa-token 安全管理
